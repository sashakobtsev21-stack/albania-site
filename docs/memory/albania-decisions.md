# Ключевые решения проекта Albania Guidebook

Запись проектной памяти (зеркало для команды). Источник правды по продукту — `SPEC.md`,
правила — `CLAUDE.md`. Здесь — устойчивые решения/договорённости, которые легко забыть.

## Гео и язык — EN-first
Аудитория Tier-1 (Запад). **EN — ведущий язык, на корне `/`**; ru → `/ru/`, uk → `/uk/` —
вторичные зеркала. Темы/ключи берём от EN-интента, ru/uk зеркалят факты. Локальный
албанский (`sq`) НЕ делаем. `DEFAULT_LANG='en'`.

## Валюта — албанский лек (ALL)
Все цены/курсы — в **леке (ALL)**. Грузинский лари/GEL вычищен (live-курс RatesBar тоже на лек,
источник ExchangeRate-API; погода — Тирана/Дуррес/Саранда + Ионическое море, open-meteo).

## Модель «город = статья»
Город-путеводитель — это **статья с `category: 'cities'`** (URL `/cities/<slug>/`), а не запись
коллекции `cities`. Коллекция `cities` для городов-путеводителей по этой модели не используется.

## Коллекция `news` — это категория, не коллекция
`export const collections` = `{ articles, routes, cities, restaurants, services }` — **`news`
там НЕТ**. Новости = статьи с `category: 'news'`, маршрут `/news/`. Фантомной коллекции/папки
`src/content/news/` быть не должно (и нет). Это согласовано с моделью эталона.

## Механика новостей 10/2 — уже в движке
Окно раздела `NEWS_SECTION_WINDOW=10` дней, окно главной `NEWS_HOME_WINDOW=2` дня, ежедневный
`daily-news-rebuild.yml` — **реализованы и работают**, чинить не нужно. Локализация была нужна
только в ТЕКСТЕ скилла `/news` (сделано: источники/аэропорты/лек/города/виза под Албанию).

## Правило дисциплины — commit + push после каждой доработки
После КАЖДОЙ доработки: синхронизировать доки (`PROGRESS`/`HANDOFF`/`AUDIT` + затронутые) →
сразу `git commit` + `git push` в `main`, **без напоминаний**. Перед push — `git pull`
(fetch + rebase), т.к. авто-джоб новостей пушит первым. Доработка без пуша не завершена.

## Стандарты публикации (объём и фото)
Статья **1200–2000 слов**; новость **1500–2000 знаков** (3–5 разделов, ≥4 внутр. ссылки).
Фото: статья **≥5** (cover + ≥4 инлайн), новость **≥2**, маршрут — cover + каждая остановка,
город **≥10**, столица (Тирана) **≥15**. Каждое фото уникально, webp ≤200 КБ, своё или CC/PD
с атрибуцией. Тон — как у носителя (uk — живая мова), не ИИ-генерёнка. Полный список — `CLAUDE.md`.

## `docs/_georgia-reference/` — намеренный шаблон, не мусор
Грузинские рабочие доки/память сохранены как **референс-шаблон**. Не удалять и не «чистить».
Не путать их содержимое с актуальными албанскими доками.
