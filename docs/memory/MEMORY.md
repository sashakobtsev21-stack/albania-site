# Memory index — Albania Guidebook

> Решения/договорённости по **албанскому** проекту. Грузинская память — референс в
> `docs/_georgia-reference/memory/` (намеренный шаблон, не удалять). Детали ключевых
> решений — в [`albania-decisions.md`](albania-decisions.md).

- [Гео и язык — EN-first](albania-decisions.md#гео-и-язык--en-first) — EN на корне `/`, ru/uk вторичны; темы от EN-интента.
- [Валюта — лек (ALL)](albania-decisions.md#валюта--албанский-лек-all) — все цены/курсы в леке; лари/GEL вычищен.
- [Город = статья](albania-decisions.md#модель-город--статья) — город-путеводитель = статья `category:'cities'`, не коллекция `cities`.
- [Коллекция news](albania-decisions.md#коллекция-news--это-категория-не-коллекция) — `news` это категория статьи (маршрут `/news/`), не коллекция; папки `content/news/` нет.
- [Новости 10/2 — в движке](albania-decisions.md#механика-новостей-102--уже-в-движке) — окна 10 дн/2 дн + ежедневный rebuild уже работают; локализован только текст `/news`.
- [Правило дисциплины commit+push](albania-decisions.md#правило-дисциплины--commit--push-после-каждой-доработки) — после каждой доработки: доки → commit → push в `main` без напоминаний.
- [Стандарты публикации](albania-decisions.md#стандарты-публикации-объём-и-фото) — объём 1200–2000 слов; фото статья ≥5/город ≥10/столица ≥15.
- [_georgia-reference = шаблон](albania-decisions.md#docs_georgia-reference--намеренный-шаблон-не-мусор) — грузинские доки намеренный референс, не мусор, не удалять.
