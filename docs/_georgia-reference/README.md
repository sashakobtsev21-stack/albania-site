# _georgia-reference — шаблоны от грузинского проекта

Это **архив рабочих документов Georgia Guidebook**, перенесённый при bootstrap'е
`albania-site` (форк движка). Здесь НЕТ движкового кода — только продуктовые
рабочие доки грузинского проекта, оставленные **как шаблоны/референс** для
аналогичных артефактов под Албанию.

Содержимое НЕ относится к Албании и НЕ должно публиковаться. Используйте как
образец структуры (план контента, брифы, аудиты, роадмап, память решений) при
наполнении албанского сайта на следующем шаге.

## Что внутри
- `CONTENT_PLAN.md` — план контента (грузинский) — образец структуры плана.
- `briefs/` — брифы статей/разделов (грузинские) — образец формата брифа.
- `AUDIT.md` — аудит сайта (грузинский) — образец чек-листа аудита.
- `PROGRESS.md`, `HANDOFF.md` — статус/снимок передачи (грузинские) — образцы.
- `ROADMAP.md`, `ROADMAP-GROWTH-2026.md` — роадмапы (грузинские) — образцы.
- `TODO-affiliates.md` — план партнёрок (грузинский) — образец.
- `GROWTH-MONETIZATION-ANALYSIS-2026-06-20.md` — анализ роста/монетизации (грузинский).
- `memory/` — авто-память грузинского проекта (решения/договорённости) — образец
  того, какие решения стоит фиксировать. Живая память Албании ведётся в
  `docs/memory/` (сброшена при bootstrap'е).

## Системные доки остались в корне (НЕ здесь)
`SPEC.md`, `CLAUDE.md`, `CONTENT_GUIDE.md`, `SETUP-GUIDE.md` — это движок/процесс,
они в корне репозитория и будут перемоделированы под Албанию на следующем шаге.
